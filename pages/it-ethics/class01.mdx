# 정보통신윤리 1주차 - 인터넷 윤리의 개념

## 전통 윤리

- 윤리 : 인간관계의 이법
  - 이법 : 어긴다고 해서 처벌받거나 하진 않음
- 도덕 : 유교적 어감이 강함. 유교적 이상이 도덕
- 예절 : 문화적 특성에 따라 적절/부적절을 판단
- 법 : 성문법 기준으로 합법/불법 판단

### 윤리관

동/서양의 윤리관은 다르다. 하지만 중요한 것은? **인간 존엄을 향한 가치관**이다.

### 자연관

서양의 자연관 : **이원론적** 사상이 주류를 이룸 (도구적 이성관이 지배하게 되며 17세기쯤부터는 달라짐)
-> 서양에서의 자연은 이용하고 지배할 수 있다.

### 윤리의 다섯가지 목적

- 사회의 분열 방지 -> **학생**의 분열 방지
- 인간의 고통 완화 -> **학생**의 고통 완화
- 인간의 행복 증진 -> **학생**의 행복 증진
- 이익 갈등을 정의롭고 질서 있게 해결
- 칭찬과 비난, 보상과 처벌 부과

정보통신윤리 교육을 학생들에게 잘 알려줘야 한다 (전체 교과서의 15%나 이 부분을 차지함)

## 인터넷 윤리

### 윤리를필요로 하는 정보화 사회의 특징

**정보 공해 현상**

- 정보통신기술의 발달이 꼭 학생들한테 좋지만은 않구나
- 테크노스트레스(technostress) - 어르신들 기술 따라가는 거 힘들어하는 것, SNS 강박증

### 정보화 사회 (사이버 공간)의 특징

- 일상에서 가능한 도덕적 규제와 통제가 어려움
  - 오프라인 세상이라면 뭔가 문제가 눈에 보일 수 있는데
  - 사이버 환경에서 발생하는 문제는 눈에 잘 보이지 않음 (사이버 폭력같은 것들) - 그래서 오히려 신체적 폭력같은 것들보다 더 위험할 수 있음

### 정보화 사회의 부작용

- 게임 : 폭력적인 온라인 게임의 영향? 근데 이건 반론도 있음 (나도 그렇게 생각함)
- 음란물 : 요즘에는 옛날보다 접근 자체가 너무 쉬워짐
- 비행 행동 : 온라인 상에서 타인을 비방한다거나, 신상 정보를 훔쳐서 사용하는 것, 저작권같은 것도 막 쓰기도 하고

### 정보화 사회의 역기능, 각종 중독 현상

- 게임 중독
- 채팅 중독
- 음란물 중독
- 검색 중독 : 요런거는 요즘 성인들한테도 문제긴 하지

### 학교에서 인터넷 윤리 교육을 하는 목적

-> 학생들이 사이버공간에서 건전한 방식으로 교류함으로써, 현실 세계의 제한된 인간관계에서 벗어나 폭넓은 인맥/지식을 쌓게 해주는 것

### 인터넷 윤리가 지향하는 3가지 자세

1. **인간 존엄**의 자세 : 보통 인터넷을 쓴다고 하면 최신 정보를 얻기 위해서 쓰는 경우가 많은데, 이런 정보를 얻는 과정에서 다른 사람의 존엄을 침해하는 행동을 하지 않아야 함 (ex. 최신 정보가 아닌 글이라고 해서 중복 게시물이라고 뭐라 한다거나 등등)
2. 자율적 책임의 자세 : 내 계정에서 일어난 일은 내가 책임진다. 이런 자세를 가지도록 지도해야 함
3. 공동체 중시의 자세 : 누구나 최소 하나 이상의 공동체에 속해 있음. 자기가 속한 공동체에 피해는 가지 않도록 (ex. 막 연락 안 한지 3~4년 지난 친구한테 게임 초대 메세지 이런 거 보낸다거나)
